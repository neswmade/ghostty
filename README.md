# nesw ghostty
## install

```bash
mv ~/.config/ghostty ~/.config/ghostty.bak
git clone https://github.com/neswmade/ghostty.git ~/.config/ghostty
```

## dependencies
 - Monaspace Neon NF (font)
 - fish (shell)

## theme

nesw themes are build in
pick one variant in `config`:

- `nesw-dark`
- `nesw-default`
- `nesw-light`

preview theme with `ghostty +list-themes`
